AmbientRealtimeData

data class AmbientRealtimeData(val date: String, val tz: String, val dateutc: Long, val macAddress: String?, val deviceId: String?, val values: MutableMap<String, Any?> = mutableMapOf())(source)

Represents the data that is sent by the Ambient Realtime API.

Note that for data, deviceId does not appear to be populated, but macAddress is, whereas for subscribed, macAddress does not appear to be populated, but deviceId is.

values contains all the data passed along (e.g. temp1f, pm25, humidityin, etc) depending on the reporting sensors
